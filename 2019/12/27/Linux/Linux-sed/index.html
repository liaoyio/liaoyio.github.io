linux 三剑客 find 、grep、sed整理

find

主要是用于查找Linux 操作系统的某个文件、目录所在路径，查找出满足条件的文件或者目录的路径.

命令格式:

  1. date: 要査找的目彖路径
    ~ 表示$HOME目录
    . 表示当前目彖
    / 表示根目彖

  2. print 表示输出结果标准输出

  3. exec 対匹配的文件抉行垓参数所给出的shell命令，形式为 command {} ; 注意{}与;之同有空格

  4. ok 与 exec 作用相同 执行命令前会提示确认是否执行

  5. |xargs 与exec作用相同,起承接作用

区別在于|xargs 主要用于承接刪除操作，而-exec都可用如复制丶移动丶重命名等。
